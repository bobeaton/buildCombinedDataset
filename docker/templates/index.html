API Endpoints

  • GET /api/v1/tts/health/
  • GET /api/v1/tts/characters/?variant=a|b
  • POST /api/v1/tts/synthesize/ — JSON {text, character, variant, compare, seed} → wav or zip
  • POST /api/v1/tts/synthesize-file/ — multipart 'file' or JSON {text, variant} → wav
